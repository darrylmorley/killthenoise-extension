KillTheNoise Settings

Blocked Keywords

Enter one keyword per line. Videos with these words in the title or description will be hidden.

Examples: "shocking", "gone wrong", "disaster", etc.

Blocked Hashtags

Enter one hashtag per line. Videos with these hashtags will be hidden.

Examples: "#shorts", "#drama", "#exposed", etc. The # symbol is optional.

Advanced Settings

When enabled, detailed logs will be shown in the browser console. This is useful for troubleshooting issues.
